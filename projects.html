My Projects

Here you can find more information about my research and my Kaggle project for the Instacart Market Basket Analysis challenge as well.

Tropical Root Polytopes Research

Code implementations for algebraic combinatorics research. The goal of this project was to evaluate the spectra of certain Tropical Root Polytopes to prove the generalized Hodge Conjecture.

Instacart Market Basket Analysis

A machine learning project using the data sets provided by the Instacart Market Basket Analysis challenge from the Kaggle platform. Here you can find code implementations for exploratory data analysis as well as the results from a few models, such as logistic regression and KNN regression.
