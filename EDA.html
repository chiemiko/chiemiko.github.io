About the Data Itself¶

There is a summary file containing data for each flight in the dataset. Then there are several datasets containing information about a singular flight. For these flight data sets, information is logged at various times starting from -5 seconds before the flight takes off to 15 seconds after the flight takes off.

*Please note! I have moved all of the csv files into a folder called 'data'. All commands in this project will refer to the csv files which are stored in this data folder.¶

What Variables To Consider???¶

The data in the summary data set includes information based off of not only the flight and Zip devices themselves, but also the weather and environmental factors during these flights as well.¶

My idea for this exploratory data analysis is to focus on these environmental factors as explanatory variables to gauge whether they have any association to the flight data and performance.¶

Organizing the Data¶

If we want to evaluate various relationships between the environmental factors recorded during the overall times of each flight, we can select just one timestamp from each flight csv in order to evaluate the records of each flight at amount of time relative to the launch time.

Considerations¶

Ideally, we would like to be able measure product performance. My knowledge about drone performance is limited... But here is what my guess is as far as what we are looking for:

  • We can look at the overall distributions of each of the columns from flight to flight... This means that across flights, if we fix a unit of relative time from launch (ie one specific time-stamped row from each flight csv), we can measure variables across columns (ie velocity or acceleration) from flight to flight
  • If there are any slower Zip devices that are not moving at all or are slower than the rest, we can assume these are "low-performing" planes

Considerations Mentioned in The Video Are:¶

link: https://www.youtube.com/watch?v=jEbRVNxL44c

  • Getting a plan that quickly into the air
  • The narrator in the video states that each Zip is designed to be able to consistently take off within 0.3 seconds... We could test whether this is really true!
